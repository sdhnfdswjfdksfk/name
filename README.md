# Training Planner

## Автор
**Широкова Виктория** 

## Описание
Training Planner - это графическое приложение для планирования и учета тренировок. Приложение позволяет добавлять тренировки, фильтровать их по типу и дате, а также сохранять данные в JSON файл.

## Функциональные возможности
- Добавление тренировок с указанием даты, типа и длительности
- Валидация вводимых данных (формат даты, положительное число для длительности)
- Фильтрация тренировок по типу и дате
- Удаление тренировок
- Автоматическое сохранение данных в JSON файл
- Загрузка данных из JSON файла при запуске

## Требования
- Python 3.6 или выше
- Tkinter (обычно входит в стандартную установку Python)
